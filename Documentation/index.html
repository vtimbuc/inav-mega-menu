A) Menu Variants - top

Variants

This menu have 4 variants:

  • Horizontal Menu
  • Stiky Footer
  • Vertical Left
  • Vertical Right

The files are organized in two folders. The first folder is the "Empty", this folder have four html files to start your iNav menu with less code. The second folder is "Examples" and have four html files, one for each variant with content examples. The CSS and HTML files are exactly the same in all menu variants, the only think that differs are the class names and the Stiky Footer Menu have 2 more divs to center the menu on page.

Colors

This menu have 11 preset difrent color schemes. To change the menu color scheme just add the respective color class to <ul class="nav blue"> .

  • Available colors:
  • blue
  • darkcyan
  • green
  • yellowgreen
  • chocolate
  • gold
  • red
  • purple
  • hotpink
  • white

B) HTML Structure - top

<HEAD> of HTML file

In the <head> of your html file you just need to add 3 links to load css files.

<!-- iNav Stylesheets -->
<link href="css/inav.css" rel="stylesheet" media="screen">
<!--[if IE 8]><link href="css/inav-ie-8.css" rel="stylesheet" media="screen"><![endif]-->
<!--[if IE 7]><link href="css/inav-ie-7.css" rel="stylesheet" media="screen"><![endif]-->

If you don't want to support IE remove the link of the respective version.

<BODY> of HTML file

Bellow is an example of the html structure, fore more info and examples take a look at the files in the "Menu Examples" folder.

<ul class="nav">

C) CSS Files and Structure - top

I'm using three CSS files in this navigation menu. The first one is a generic file. Many browser interpret the CSS code differently and the browser who most differe is the Internet Explorer, so I've created 2 more css files to make the menu compatible with IE8 and IE7. The IE6 is a very old browser and is hard to make this menu work in this version, so IE6 it's not supported.

The file is separated into section:

/**************************************************

	01. RESET & FONTS & OTHERS
	02. MENU GENERAL STYLING
	03. DROPDOWN STYLING & LAYOUT
	04. SEARCH FORM
	05. LOGIN FORM
	06. CONTENT STYLING
	07. STIKY FOOTER
	08. VERTICAL MENU
	09. VERTICAL MENU RIGHT
	10. EXTRA COLORS

**************************************************/

If you would like to edit a specific section of the menu, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

D) Contact Form - top

To create you working contact form add this html to the menu:

<!-- Contact Item | Start -->

<li class="contact-nav"><a href="#" class="nav-item drop">Contact</a>

	<div class="drop-form left">

		<div class="inav-cffb"><div class="inav-alert"></div></div>

		<form action="contact/process.php" method="post" id="inav-cform">
			
			<input type="text" name="name" id="name" placeholder="Your Name">
			<input type="email" name="email" id="email" placeholder="email@email.com">
			<textarea name="message" id="message"></textarea>
			<input type="submit" value="Send Message">

		</form>

		<hr>

		<ul class="social">
			<li><a href="#"><img src="img/social/facebook.png" alt="Facebook"></a></li>
			<li><a href="#"><img src="img/social/twitter.png" alt="Twitter"></a></li>
			<li><a href="#"><img src="img/social/rss.png" alt="RSS Feed"></a></li>
			<li><a href="#"><img src="img/social/vimeo.png" alt="Vimeo"></a></li>
			<li><a href="#"><img src="img/social/dribbble.png" alt="Dribbble"></a></li>
		</ul>

	</div><!-- end .drop-normal-box -->

</li>

<!-- Contact Item | End -->

Then add this two JavaScript links before the ending of your body tag and edit the "config.php" file.

<!-- Contact Form JS -->
<script type="text/javascript" src="contact/jquery-1.8.3.min.js"></script>
<script type="text/javascript" src="contact/jquery-form.min.js"></script>
<!-- end Contact Form JS -->
